src.Game.GameProgram¶

Handles game loops, key presses, and shutting down.

Holds the active state.

Tells state to update whenever its frame rate time passes

static get_game_thread_cb(game_program)¶

Callback to get a game thread.

Returns a function to attach to a game thread. Game thread should not be a daemon. The game thread loops while program.running is true. If the time since last update exceeds program.frame_rate, thread calls program.update(time interval).

static on_key_cb(program, key_type)¶

return callback passed to pynput keyboard listener.

Builds a dictionary based on key presses / releases and sends that dictionary to program.execute when a key press is read.

Called twice, once to get a callback for key presses, once to get a callback for key releases.

Parameters

  • program (class GameProgram) – Program where key presses are read.

  • key_type (str) – Either ‘press’ or ‘release’
