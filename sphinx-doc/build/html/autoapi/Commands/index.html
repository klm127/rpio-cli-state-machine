Commands¶

Classes for binding conditions to effects on the state of the program, using callback functions for input testing and state effect alteration.

class Commands.Command(state, bool_func, effect, name='untitled')¶

Binds an evaluation callback to an effect callback.

Parameters

  • state (class State) – The state using this Command

  • bool_func (Function) – A callback which returns True if the effect should be triggered

  • effect (Function) – A callback called when command should be executed

  • name (str) – Optional parameter containing human-readable command name

check_command(self, test)¶

Checks test with self.bool_func(self.state, test) callback. If true, calls self.effect(self.state, test)

Parameters

test (str | dict) – Input to test.
